Sistema de Gestión de Biblioteca Digital

Este proyecto es un sistema de gestión de biblioteca desarrollado en C++ con enfoque en programación orientada a objetos. 
El objetivo es modelar y administrar elementos como libros, usuarios y préstamos, integrando conceptos fundamentales de POO: 
clases, atributos, métodos, herencia, adicion y diagramas UML.

El sistema permite registrar libros, gestionar usuarios y controlar préstamos activos, además de servir 
como práctica para representar y analizar relaciones entre objetos mediante diagramas de clase y de secuencia.
